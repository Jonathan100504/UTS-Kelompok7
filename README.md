Perancangan dan Implementasi Sistem Analisis Sentimen Ulasan Aplikasi Google Play Store Berbasis IndoBERT dan BERTopic

Proyek ini merancang dan mengimplementasikan sistem analisis sentimen ulasan aplikasi Google Play Store berbahasa Indonesia menggunakan model IndoBERT untuk klasifikasi sentimen dan BERTopic untuk analisis topik, yang mencakup proses pengumpulan data otomatis, preprocessing teks, pelatihan model deep learning, serta visualisasi hasil dalam antarmuka web berbasis Flask.
